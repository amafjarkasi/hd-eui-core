# Security Policy

## Supported Versions

We support security updates for the latest major version of HD-EUI Core. Use the following table to see which versions are currently supported:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in HD-EUI Core, please report it responsibly. Do not create a public GitHub issue for security matters.

1. **Contact Us**: Email security reports to [your-email@example.com] (replace with your actual email) with details including:
   - Description of the vulnerability.
   - Steps to reproduce.
   - Affected versions.
   - Potential impact.

2. **Response Time**: We aim to acknowledge reports within 48 hours and provide a detailed response within 7 days, including a fix timeline if applicable.

3. **Disclosure Policy**: We follow a 90-day disclosure policy. After confirmation, we'll work on a fix and coordinate disclosure with you.

4. **Bounty**: We do not currently offer a bug bounty program, but we appreciate and credit responsible disclosures in release notes.

Thank you for helping keep HD-EUI Core secure!

For non-security issues, use GitHub Issues: https://github.com/amafjarkasi/hd-eui-core/issues
