The High-Density Enterprise UI (HD-EUI) design system is optimized for data-dense, professional B2B interfaces.
Design Philosophy:
- Hyper-Density: Prioritize information density over whitespace. Tight gaps (2px-4px) and minimal padding (p-1).
- Structured Layout: Use hard 1px borders (#d8dcde) for sectioning rather than shadows.
- Contextual Sizing: Designed for sidebars (300px-380px) or complex dashboards.

Core Design Tokens:
- Color Palette: Primary #2f3941 (Slate), Hover #1f252a, Borders #d8dcde (Gainsboro), Brand/Focus #1f73b7 (Azure), Muted #87929d.
- Typography: Headers 11px font-bold, Body 10px, Muted 9px, Micro 8px.
- Spacing: Tight 2px, Ultra 4px, Micro 1px.
</context>
